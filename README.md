# 🎬 CineSeek – Movie Search Web App

CineSeek is a modern **movie search web application** that allows users to quickly search for movies and view detailed information such as ratings, release date, plot, and cast.

The application fetches **real-time data from a movie API** and displays it in a **clean and responsive interface**. It is built using **React.js and modern frontend tools**.

---

## 🚀 Features

- 🔎 **Movie Search** – Search movies instantly by title  
- 🎞 **Detailed Movie Information** – View plot, genre, cast, ratings, and release date  
- 📱 **Responsive Design** – Works on desktop, tablet, and mobile  
- ⚡ **Fast UI** – Built with modern frontend tools for performance  
- 🌐 **API Integration** – Fetches real movie data from an external API
